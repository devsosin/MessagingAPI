# Messaing API (Send Email, Kakao Talk, SMS)

## Providers

- Eamil Providers
- Solapi
- Aligo

## Environment Variables

**ㅁ for sending email**

```
EMAIL_SERVER=
EMAIL_USERNAME=
EMAIL_PASSWORD=
```

**ㅁ for send Alimtalk (Solapi)**

```
SOLAPI_API_KEY=
SOLAPI_API_SECRET=
SOLAPI_PF_ID=
SOLAPI_SENDER_NUMBER=
```

**ㅁ for sending sms (ALigo)**

```
ALIGO_API_KEY=
ALIGO_USER_ID=
ALIGO_SENDER_NUMBER=
ALIGO_TEST=Y/N
```
